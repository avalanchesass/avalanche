# avalanchesass_object_island
Box off content.

## Install
```
npm install avalanchesass_object_island --save
```

## Basic usage
```css
/* Import the main file. */
@import 'avalanchesass_object_island';

/* Import just the mixin file. */
@import 'avalanchesass_object_island/mixin';
```
